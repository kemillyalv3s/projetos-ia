## Subi um repositório novo

> * 1. iniciando um projeto
```
git init
```

> * 2. adicionar na area de stage
```
git add .
```

> * 3. Criando o copiar o comando no github do repo
```
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/kemillyalv3s/projetos-ia.git
git push -u origin main
```
